Getting Started

Use the top navigation to open Games, Proxies (browsing), Blooket, Docs, and Issues.

Some websites will not work perfectly. If something breaks, post it in Issues with a screenshot and what you tried.

Browsing

Open /browsing (Proxies). Enter a URL (example: youtube.com) or a search query. The site will open it in the built-in browser.

Tips

  • If a page looks unstyled, refresh the page.
  • If a site blocks loading (403/406), it is usually the site blocking this browsing method.
  • If media does not play, reload the page and try again.

Tabs And URL Bar

The built-in browser supports multiple tabs. The URL bar always shows the current URL for the active tab. You can copy it, edit it, and press enter to navigate.

Refreshing

  • Use the page refresh button (or Ctrl+R).
  • If the browser gets stuck, close the tab and open the site again.

Proxy Engines

On /browsing, Wubu uses a default engine for compatibility and will fall back automatically if something fails to load. There is not a manual engine switch on that page yet.

Settings

Click the gear icon in the top right to open settings. If you change settings and a site stops working, switch back to defaults and reload.

  • Tab Appearance: change the tab title and icon (includes a Clever preset).
  • Advanced: connection and other options.

Games

Open /games to browse the game library. Use the search bar to find a game fast, then open it from the directory. Most games open in an in-page player.

Blooket

Open /blooket for the Blooket page. It streams a remote browser view so Blooket runs more reliably than normal proxy modes. It can use more data/CPU and may feel a little delayed.

Issues Board

Open /issues to report problems. Posting is anonymous (no account).

What you can do

  • Create an issue with a title and details.
  • Attach screenshots (optional).
  • Reply to an issue to add more info.
  • Like or dislike issues so the most important ones float to the top.

How to write a good report

  • Site: example youtube.com
  • Engine: Ultraviolet, Scramjet, or Rammerhead
  • What happened: include the exact error text if possible
  • Steps: what you clicked/typed to make it happen
  • Screenshot: include the full page if you can

Privacy And Safety

  • Do not post personal information in Issues.
  • Do not upload screenshots that include passwords, emails, or private messages.
  • Some websites may detect or block this browsing method.

Troubleshooting

Page is blank or looks broken

  • Refresh the page.
  • Open the site again in a new tab.

403 Forbidden / 406 Not Acceptable

This is usually the website blocking access. Try a different engine or try again later.

Issues page: posting fails or shows 503

The Issues service may be temporarily unavailable. Refresh and try again later. If it keeps happening, report it to the site owner.
