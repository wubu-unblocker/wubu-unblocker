This web based proxy is based on scramjet. It uses advanced web proxying techniques to provide free web usage for users.

## Hugging Face Space deployment (no zip)

Push the repository directly to your Space instead of uploading an `app.zip`:

```bash
HF_TOKEN=hf_xxx ./scripts/deploy-hf-space.sh wubu-unblocker/v1.0
```

This pushes your current git `HEAD` to the Space's `main` branch.
